# Install Compilers

C/C++
```
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt-get update
sudo apt-get install g++-4.9


sudo ln -f -s /usr/bin/g++-4.9 /usr/bin/g++

```

Java
```
sudo add-apt-repository ppa:openjdk-r/ppa  
sudo apt-get update   
sudo apt install openjdk-8-jre
```
